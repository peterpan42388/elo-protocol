# ADR-005: Ranking Governance Minimum

- Status: Accepted
- Date: 2026-03-07

## Context
Ranking must reward quality while limiting manipulation.

## Decision
Use minimum ranking factors in P3:
- relevance score
- listing rating
- provider rating
- token-saving evidence
- reliability (SLA/uptime)
- price sensitivity

Anti-abuse baseline:
- only-post-usage reviews
- owner-level rate limiting for reviews
- suspicious cluster down-weighting

## Consequences
- Ranking stays explainable.
- Provides baseline fairness before DAO governance expansion.
