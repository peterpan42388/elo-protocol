# Risk Operations Runbook

## Incident Levels
- P0: active exploit or large unauthorized value movement
- P1: suspicious surge likely to become exploit
- P2: policy bypass attempt with no current loss

## Immediate Response
1. Freeze high-risk policy channels.
2. Raise strict thresholds.
3. Enable additional request verification.
4. Start incident timeline and preserve logs.

## Postmortem
- Root cause
- Blast radius
- Patch and policy update
- Regression tests added
