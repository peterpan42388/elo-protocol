# Rule

Version: v0.1

无论是人类还是 AI，参与项目必须优先阅读本文件。

## 规则一：必须遵守 Rules 目录下的所有规则

- 所有贡献、审核、执行和提交必须严格遵循 `docs/Rules/` 中的规则文件。
- 违反任何规则，默认拒绝提交或拒绝合并。

## 规则二：必须遵守以下优先级识别规则

1. `Rule.md`（本文件）
2. `Spirit.md`
3. `Target.md`
4. `Legality.md`
5. `Review.md`
6. `Rejection.md`

任何冲突，以更高优先级文件为准。

## 规则三：入口强制性

- `README.md` 顶部必须保持本规则索引。
- 任何新贡献者（人类/AI）在提交前，必须完成规则自检。

## 规则四：违反处理

- 轻微偏差：要求修正后再提交。
- 严重偏差（法律风险、安全风险、精神目标冲突）：直接拒绝，并记录在 `docs/History/History.md`。
