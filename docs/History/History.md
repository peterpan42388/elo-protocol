# History

Version: v0.1

本文件记录项目版本历史与关键阶段演进。后续每次阶段跃迁或重大规则变更，都应在此追加记录。

## v0.1（2026-03-08）
- 项目启动并定义核心结算规则：
  - `same_owner_free`
  - `cross_owner_paid`
- 建立基础 docs、contracts、tests、api scaffold。
- 焦点：AI 代理经济可持续性验证。

## v0.2（P2）
- Solidity 合约、测试网部署脚本、基础风控插件机制到位。
- 增强协议可验证性和可审计性。

## v0.3（P3）
- 引入 x402 / ACP 适配层与市场 API。
- 建立人类控制台与交易评价基础流程。

## v0.4（规则体系强化）
- 建立 `docs/Rules` 与 `docs/History` 体系。
- 引入项目强制规则入口（Rule-first）。
- 明确“Target 作为导向”的阶段迁跃。

## v0.5（OSCP 方向冻结）
- 将 `Open Simulation Commons Protocol (OSCP)` 确认为下一阶段核心方向。
- 明确 `虚拟未来` 作为模拟世界昵称。
- 冻结 OSCP 的世界模型、协议层、技能层与阶段路线图。
- 明确平台不承担中心化实现成本，需求优先转交给用户自己的 Agent 执行。

## 维护要求

每次更新请附带：
1. 时间与版本标签
2. 变更摘要
3. 风险与回滚说明
4. 对项目目标（Target）的影响评估
