# MindJourney

Version: v0.1

本文件记录项目从构想到当前阶段的思路演进，核心转折点为“Target 作为导向”。

## 初始阶段：协议基础构想

- 项目起点是构建 AI Agent 之间的开源结算框架（ELO）。
- 早期重点是证明：AI 不只是任务执行器，也可以成为经济协作主体。
- 核心问题：如何在不牺牲开放性的前提下，建立可持续交易规则。

## 中期阶段：市场与生态扩展

- 扩展到 Agent 市场视角：API / Skill / Workflow 的可检索与可交易。
- 识别痛点：传统支付路径对高频 Agent 协作存在摩擦与延迟。
- 引入：凭证链、批结算、评价机制、风控插件等设计。

## 关键转折点：Target 作为导向

- 项目定位从“技术可行性”升级为“社会导向型开源基座”。
- 目标明确为：
  - 避免 AI 生产力被极少数主体垄断；
  - 帮助普通人在 AI 阵痛期拥有基本生存与服务接入能力；
  - 通过开源重构服务体系，降低结构性冲突风险。

## 当前阶段使命

- 持续完善规则系统（Rules）与历史可追溯体系（History）。
- 把工程决策与项目精神、目标、法律边界强绑定。
- 为下一阶段“项目基座打造”提供统一准则和执行边界。
